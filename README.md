🏅 Medalhistas das Paralimpíadas de Paris 2024
Este projeto apresenta uma análise detalhada do desempenho dos atletas brasileiros nos Jogos Paralímpicos de Paris 2024.
O objetivo é transformar dados brutos em insights estratégicos sobre biometria, distribuição geográfica e performance por modalidade.

🚀 Objetivo do Projeto
Desenvolver um dashboard interativo no Power BI que permita ao usuário explorar o perfil dos atletas medalhistas e entender os fatores que contribuíram para o sucesso da delegação brasileira.

🛠️ Tecnologias Utilizadas

Microsoft Excel: Utilizado como a fonte de dados primária do projeto: estruturação de tabelas relacionais e tratamento de dados (ETL).

Power BI: Construção dos visuais e dashboards interativos.

DAX (Data Analysis Expressions): Criação de medidas personalizadas e inteligência de tempo.

Power Query: Limpeza, transformação e tipagem dos dados.

Power Point: Design do background personalizado (UI/UX).

📊 Funcionalidades e Análises
O dashboard está dividido em quatro visões principais:

Início: Menu de navegação intuitivo para acesso às diferentes áreas do relatório.

Panorama Geral: Panorama dos principais KPIs, como total de medalhas (Ouro, Prata e Bronze) e países medalhistas.

Perfil dos Atletas CPB: Perfil individual detalhado com fotos dinâmicas, biografia, idade, peso e altura de cada medalhista brasileiro.

Estatísticas CPB: Estudo da distribuição etária, por gênero e correlações geográficas (estados que mais geraram medalhas).

💡 Principais Insights
Pico de Performance: A maior concentração de medalhistas está na faixa etária entre 25 e 35 anos.

Dominância Regional: O estado de São Paulo lidera o ranking de medalhas, seguido por estados com fortes centros de treinamento paralímpico.

Equilíbrio de Gênero: Análise visual da paridade de medalhas conquistadas por homens e mulheres em modalidades como Atletismo e Natação.

📂 Como visualizar o projeto
Baixe o arquivo .pbip (deixei um .pbix também) deste repositório.
